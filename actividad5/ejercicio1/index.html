Etiquetas HTML

<dl>

El elemento dl representa una lista descriptica. Este elemento encierra una lista de grupos de términos y de descripciones. Algunos usos comunes para este elemento son implementar un glosario o para desplegar metadatos.

<dt>

Este elemento especifica un término en una descripción o una lista de definiciones, y como tal debe utilizarse dentro del dl y seguido por un elemento dd, sin embargo, múltiples elementos dt en un renglón indican diferentes términos los cuales todos son definidos por el siguiente elemento dd.

<dd>

El elemento dd provee detalles acerca de o la definición de un término precedente (dt) en una lista de descripciones (dl).

Los principales navegadores web instalados en ordenadores

Google Chrome
Google Chrome es un navegador sencillo, seguro y rápido gracias a las funciones inteligentes integradas de Google. Además de dichas funciones que te ayudarán a hacer todo lo que quieras (búsquedas rápidas, traducciones con un solo clic, artículos personalizados…)
Microsoft Edge
Microsoft Edge es el único navegador con herramientas integradas que te ayudan a ahorrar tiempo y dinero cuando compras online. Posee un gran rendimiento, compatibilidad y velocidad para que navegar por la web sea aún más fácil.
Firefox
Firefox es un navegador web que protege tu privacidad, y es que no tiene políticas sospechosas u otras condiciones en las cuales se prioricen los intereses de los anunciantes. Además, al utilizar herramientas que no invadan tu privacidad, la navegación por las páginas web es más rápida.

<blockquote>

Permite a los autores insertar citas en forma de bloques de contenido, usualmente compuestas por un párrafo, un grupo de párrafos o un conjunto de muchos otros elementos incluyendo imágenes, tablas y artículos, entre otros.

Por ejemplo:

Mark dijo el 04/04/2014

No creo que esto sea un cambio importante en el juego. De todos modos, si hay un cambio, también puede ser para mejor...

Debes estar bromeando Mark! Esto destruirá a la comunidad...

<q>

Este elemento es el hermano menor de blockquote. Este elemento está destinado a citas breves que no requieren saltos de párrafo.

Por ejemplo:

Este sitio web ha sido contruido según el lema de Tim Berners Lee El poder de la web está en su universalidad. El acceso por parte de todos sin importar la discapacidad es un aspecto esencial.

<cite>

Marca una referencia a una fuente, o el autor de un texto citado. Por ejemplo para mencionar a una persona o cosa en una conversación o escrito.

Por ejemplo:

Firefox 1.0 fue lanzado en 2004 y se convirtió en un gran éxito.

<abbr>

Este elemento representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura. Si está presente, el atributo title debe contener la descripción completa y nada más.

Por ejemplo:

La ISO es el organismo encargado de promover el desarrollo de normas internacionales de fabricación (tanto de productos como de servicios), comercio y comunicación para todas las ramas industriales.

<address>

Provee información de contacto para el autor de la sección o documento, como una dirección postal, una dirección de correo electrónico, un número telefónico, una ciudad, un país, un código postal, etc.

<sub>

Este elemento representa un subíndice.

Por ejemplo:

Encuentra Rin y Rout del Transistor de unión bipolar.

<sup>

Este elemento representa un superíndice.

Por ejemplo:

(a+b)2 = a2+2ab+b2

<code>

Este elemento representa a un fragmento de código de computadora, que puede ser cualquier texto legible para computadoras.

Por ejemplo:

La función siguiente muestra un mensaje al usuario:

function mostrarMsj(msj)
{
alert("El mensaje es: "+msj);
mostrado = true;
}

Pero para que funcione correctamente necesitas declarar primero la variable global "var mostrado = false;".

<pre>

Representa un bloque de texto preformateado donde, en contraste con otros elementos, los espacios continuos y los quiebres de línea son respetados. Este elemento es comúnmente utilizado para mostrar código de computadora, texto plano o arte ascii.

Por ejemplo: 
        
        
        class GFG
        {
            // Program begins with a call to main()
            // Print "Hello, World" to the terminal window
            public static void main(String args[])
            {
                System.out.println("Hello, World");
            }
        }

<var>

Representa a una variable, siendo ésta una variable en una fórmula matemática o un código de computadora, una constante, un parámetro en una función, etc.

Por ejemplo:

el bucle no se detendrá hasta que F sea igual a 0.

<kbd>

Representa una entrada del usuario que puede ser hecha por teclado o cualquier otro medio, como mouse, pantalla táctil o comandos de voz.

Por ejemplo:

Presiona Ctrl y mantenla presionada mientras presionas C. Luego pasa al segundo documento y presiona Ctrl y mantenla presionada mientras presionas v.

<samp>

Se utiliza para incluir texto en línea que representa una muestra (o cita) de la salida de un programa de ordenador.

Por ejemplo:

Cuando ingresé el comando, la computadora mostró Error: comando no encontrado

<time>

Este elemento representa fechas, horas, desplazamientos de zonas horarias y duraciones.

Por ejemplo:

la fecha es 2022-09-08
